# Core Vocabulary — The 500 Words That Carry You Everywhere

> *"Research suggests the top 1,000 words in any language cover about 85% of everyday speech. Learn the top 500 and you're already more than halfway there."*

This isn't a random word dump. Every word here was chosen because it appears constantly in daily Japanese life — on signs, in conversation, on menus, in news. Master these and almost every real-world situation becomes navigable.

| Japanese | Kanji | Day | Memory trick |
|---|---|---|---|
| 月曜日 | getsuyōbi | Monday | 月 = moon |
| 火曜日 | kayōbi | Tuesday | 火 = fire |

### Months
January = 一月 (ichigatsu), February = 二月 (nigatsu) ... December = 十二月 (jūnigatsu). Simple: number + 月.

## Asking for Help (Survival Phrases)

| Japanese | Romaji | Meaning |
|---|---|---|
| 〜はどこですか？ | ~wa doko desu ka? | Where is ~? |
| 〜をください。 | ~wo kudasai. | Please give me ~. |
| 〜はありますか？ | ~wa arimasu ka? | Do you have ~? |
| いくらですか？ | ikura desu ka? | How much is it? |
| 分かりません。 | wakarimasen. | I don't understand. |
| もう一度お願いします。 | mō ichido onegaishimasu. | Please say that again. |
| ゆっくり話してください。 | yukkuri hanashite kudasai. | Please speak slowly. |
| 日本語が少しだけ話せます。 | nihongo ga sukoshi dake hanasemasu. | I can speak a little Japanese. |
| 英語を話せますか？ | eigo wo hanasemasu ka? | Can you speak English? |

| 道に迷いました。 | michi ni mayoimashita. | I'm lost. |
